resultFlow

fun <T> resultFlow(block: suspend ResultFlowScope<T>.() -> Unit): Flow<AsyncResult<T>>

Creates a Flow of AsyncResult with a DSL that provides explicit success, error, loading, and notStarted emissions.

Unlike the result DSL which produces a single AsyncResult, this builder produces a Flow that can emit multiple AsyncResult values over time.

Uncaught exceptions (except CancellationException) are caught and emitted as Error.

Example:

val userFlow: Flow<AsyncResult<User>> = resultFlow {
    loading()
    val user = fetchUser(id)
    success(user)
}
